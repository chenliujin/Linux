Because ssh always validate host key and then create known_hosts file under ~/.ssh directory. You can manually create the .ssh directory and chown to www-data. If not just tell ssh to skip host key checking.

rsync -e "ssh -o StrictHostKeyChecking=no,UserKnownHostsFile=/dev/null" --other-options-here
